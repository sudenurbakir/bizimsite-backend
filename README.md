# BizimSite Backend

BizimSite Backend, apartman ve site yönetim sistemi için geliştirilecek backend uygulamasının tasarım ve geliştirme sürecini içeren eğitim amaçlı bir projedir.

Bu repo, yalnızca kod geliştirme sürecini değil; aynı zamanda proje mimarisi, API tasarımı, veritabanı planlaması, iş akışları ve teknik dokümantasyon çalışmalarını da içermektedir.

---

# Projenin Amacı

Bu proje;

- Backend geliştirme sürecini öğrenmek,
- Yazılım mimarisi oluşturmak,
- API tasarlamak,
- Veritabanı modellemek,
- Teknik dokümantasyon hazırlamak

amacıyla geliştirilmektedir.

---

# Dokümantasyon

Projeye ait teknik dokümanlar `docs` klasörü altında yer almaktadır.

Dokümanlar arasında;

- Proje mimarisi
- API tasarımı
- HTTP ve REST
- Veritabanı tasarımı
- Kimlik doğrulama
- Hata yönetimi
- Loglama stratejisi
- Validasyon stratejisi
- İş akışları

gibi konular bulunmaktadır.

---

# Proje Durumu

Bu proje aktif geliştirme aşamasındadır.

Kod geliştirme sürecinden önce sistem tasarımı ve teknik dokümantasyon çalışmaları hazırlanmaktadır.
